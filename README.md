# UECS DB Access Web API

UECS DBへアクセスするためのWeb API

** Version 0.00 **

## 対象DB

 volga上の uecs0

## 機能一覧

|     名称     |             機能概要               | 状況 |
|:------------:|:-----------------------------------|:----:|
| getdata      | 日付をキーにデータを取得する       |  未  |
| getdatabyccm | CCMtypeをキーにデータを取得する    |  未  |
| getdatabyip  | IPアドレスをキーにデータを取得する |  未  |
| listip       | IPアドレスの一覧を取得する         |  未  |
| listccm      | CCMtypeの一覧を取得する            |  未  |

## 機能詳細

### getdata

 日付をキーにデータを取得する。


### getdatabyccm 

 CCMtypeをキーにデータを取得する。

### getdatabyip

 IPアドレスをキーにデータを取得する。


### listip

 IPアドレスの一覧を取得する。


### listccm

 CCMtypeの一覧を取得する。
